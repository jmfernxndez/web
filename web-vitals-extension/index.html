Web Vitals Extension

Measure metrics for a healthy site

This is a continuation of the official Web Vitals extension originally developed by Google. It measures the three Core Web Vitals metrics (LCP, FID, CLS) in a way that matches how they're measured by Chrome and reported to Google tools like Chrome User Experience Report, PageSpeed Insights, and Search Console.

As of January 2025, Google discontinued support for the original extension. This fork is maintained by jmfernandez.dev to keep it available for developers and SEO professionals.
