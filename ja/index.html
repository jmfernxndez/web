自己紹介

私は、クリーンで効率的な Web エクスペリエンスの構築に情熱を注ぐフロントエンド開発者です。主に Astro やバニラ JavaScript などのモダンなツールを使用していますが、WordPress や Shopify、インフラ構築など、プロジェクトに応じて柔軟に対応します。

現在はデジタルエージェンシー「Esparta Digital」で eコマースおよび CMS プロジェクトのフロントエンド開発をリードしています。また、VANTAG.es というオープンソースプラットフォームを運営し、新しい Web 技術を試しながら開発者コミュニティと知識を共有しています。
